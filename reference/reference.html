  4. Minimum Requirements Features

    1. HTML5

      1. Appropriate HTML5 tags, including use of HTML5 structural tags, <a>, <p>, <hn>, <blockquote>, <ol>, <ul>, <dd> to name a few.
      2. Inline images both static and clickable.
      3. Use of appropriate special characters.
      4. Use of appropriate <meta> tags.
      5. A footer on the home page indicating when the web site was last updated or modified.

    2. CSS3

      1. Use CSS3 to format the appearance of paragraphs, lists, headers, background images, tables, forms.
      2. Use CSS3 contextual selectors (e1, e2, e3; e f; e > f; e + f) and at least two attribute selectors, excluding class and id attribute selectors.
      3. Use of CSS3 styles for color (RGBA & HSLA).
      4. Use of dynamic pseudo-classes to highlight hyperlinks states of link, visited, active, hover; focus and two structural pseudo-classes: first-of-type and last-of-type.
      5. Use of CSS3 background styles (background size, background-clip, background-origin, padding, margins, borders, use of an appropriate border style, use of rounded corners).
      6. Your home page should have a drop cap created with CSS3.
      7. Define document content with the class and id properties and create styles for them.
      8. Use cascading styles to control page layout by using either grid, flex, float, positioning. The use of framework for layout is discouraged due to too many unknown factors.
      9. Use of CSS3 text and box shadows.
      10. Use of CSS3 transform rotate feature.
      11. Use of CSS3 opacity applied to two images to appear semi-transparent.

    3. Web Form

      1. Use of appropriate form elements including some of the new HTML5 form elements.
      2. Apply appropriate styles to Web Forms.
      3. Apply inline form validation by using the focus pseudo-class to appropriate form elements.

    4. Graphical Web Tables

      1. Use of graphical web tables to depict relevant data that is appropriate to the business.

    5. Multimedia Minimum

      1. Use sound, video somewhere on your site. Ensure these are of short duration preferably about 30 seconds.
      2. Use of transitions and animations.

    6. JavaScript

      1. Use of variables, arithmetic, comparison and logical operators, subroutines, functions, arrays, conditional statements, conditional operators, running time commands with system clock.
      2. Use of system date (example: clock, countdown timer to an event) on your web site.
      3. Validating form input by using appropriate JavaScript functions.
      4. Use of JavaScript alerts for validation purposes.
      5. Use of BOM/DOM in emulating events through the use of event handlers.

    7. Responsive Design

      1. Your web site design should implement each of the following responsive techniques on different pages on your web site:
        1. CSS Grid Layout using the display: grid
        2. CSS Flex Layout using the display: flex
        3. CSS Positioning.
      2. The use of contextual selectors to control sub level menus, including the use of a navicon image for mobile design is required.
